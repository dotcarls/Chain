    ValidationBuilder

    A ValidationBuilder is used to create an extensible Validation.

    An instance of ValidationBuilder will have prototype methods that are created through the register() method. When the fork() method is called all of the instance's already registered methods will be transferred to a new instance of ValidationBuilder that is then returned. This is what allows you to 'extend' a Validation as you can then continue to register() methods without mutating the original ValidationBuilder's prototype.

    These prototypical methods are called Invokable's. An Invokable is a named function reference that optionally includes some predefined arguments that can be used to make specific comparisons. *
